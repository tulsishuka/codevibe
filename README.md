Dashboard Authentication Project
Overview

This project is a full-stack authentication system built with React (frontend) and Express/MongoDB (backend). Users can register using their username, email, and password, log in securely, and access a protected dashboard.

The dashboard displays an overview of the userâ€™s activity and includes a short introduction about Brihat Infotech, demonstrating integration of informative content with a functional authentication system.

Tech Stack
```
Frontend: React, Tailwind CSS

Backend: Node.js, Express

Database: MongoDB (Mongoose)

Authentication: JWT, bcrypt
```
